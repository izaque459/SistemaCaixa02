# SistemaCaixa02
Descreve um sistema de caixa de um mercadinho
O sistema deverá ter os seguintes requisitos:
• O caixa pode processar várias compras e fica aberto até a operadora sinalizar que o
expediente encerrou;
• Cada cliente pode comprar vários itens e ao ler cada item deve ser informada a descrição do
item, a quantidade compra e o valor da unidade;
• Ao final da leitura e contabilização de cada item deve ser exibida uma mensagem para a
operadora do caixa perguntando se há mais itens a serem processados;
• Ao final da compra o cliente deve informar a forma de pagamento (1- Dinheiro, 2- Cartão de
Crédito); caso seja em dinheiro, haverá 5% de desconto no total da compra;
• Após a contabilização final da compra, deve ser exibida mensagem para a operadora de caixa
perguntando se o dia foi encerrado;
• Após o encerramento do dia o sistema deverá exibir: total geral das compras, total das
compras com dinheiro, total das compras com cartão de crédito, valor total de descontos
concedidos para os clientes.
